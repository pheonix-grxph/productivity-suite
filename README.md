🧠 Productivity Suite

A fully client-side productivity suite built in React — featuring a Pomodoro timer, task manager with filtering & sorting, habit tracker with streaks, Google Calendar-style view, and a multi-period analytics dashboard. No backend required — all data persists via localStorage.

✨ Features
⏱ Pomodoro Timer

Customizable sessions: 20 min → 3 hours
Activity mode labels — Sports, Working, Learning, Academic, Meditation, Reading
Add & manage your own custom labels
Animated ring progress with gradient glow
Wave animation on start — UI transitions into focus mode
Single Give Up button to reset mid-session
All data persists across sessions via localStorage

✅ Task Manager

Add tasks with priority (High / Medium / Low), category, due date, and time of day
Time modes: All Day, Morning, Afternoon, Evening, or Custom time range
Filter by category, priority, and date range (Today / Tomorrow / This Week / Custom)
Sort by date, priority, or category
Add custom categories on the fly
Completion tracking with visual indicators

🔥 Habit Tracker

Track daily habits with streaks 🔥
Set frequency: Everyday, Once a week, Twice a week, 3x a week, Weekdays, Weekends
Choose specific days for weekly habits
Add categories and start dates
Time of day support (Morning / Afternoon / Evening / Custom)
Habits dim automatically when not due today

📝 Notes

Quick timestamped notes
Persistent across sessions

📅 Calendar

Full monthly grid view with dot indicators for tasks (🟣) and habits (🟡)
Click any date to expand a Google Calendar-style hourly timeline (12 AM – 11 PM)
Add tasks directly from any date on the calendar

📊 Dashboard

Time-period selector: Today / This Week / This Month / Past 3 Months / This Year
Dynamic activity summary (tasks done, habits completed, focus sessions)
Progress bars for task completion, habits, and notes
Tasks broken down by category
Circular completion ring with percentage
All-time stats: best streak, high-priority remaining, total habits

🎨 Animated Backgrounds

5 live canvas animations: Rain, Stars, Aurora, Fire, Bubbles
Upload your own image or video as background

🗂 Draggable Navbar

Drag anywhere on screen
Collapse to icon-only mode or expand to show labels
All panels open as floating, draggable, resizable windows

🚀 Getting Started
This project is a single HTML file — no build step, no dependencies to install.
Run Locally
bash# Clone the repository
git clone https://github.com/pheonix-grxph/productivity-suite.git

# Navigate into the folder
cd productivity-suite

# Open in browser (just double-click or serve it)
open index.html
Or serve it with any static server:
bashnpx serve .
Deploy Your Own

Fork this repo
Go to app.netlify.com/drop
Drag and drop index.html
Done — your own instance is live! ✅
